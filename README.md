Ce dépôt contient l'ensemble de mes travaux pratiques pour le cours de Fondamentaux de l'IA. 

Chaque TP est organisé sur sa propre branche pour une consultation propre et isolée.

## 📂 Structure du Dépôt

- **Branche `main`** : Vue d'ensemble avec tous les dossiers.
- **Branche `tp1`** : Analyse de l'algorithme de Machine Learning (Iris).
- **Branche `tp2`** : Prédiction du Churn Client (IBM Telco).
- **Branche `tp3`** : Deep Learning & Vision par Ordinateur (Fashion-MNIST).

## 🚀 Comment naviguer ?

Pour accéder au code et aux livrables d'un TP spécifique, utilisez la commande `git checkout` :

```bash
# Pour voir le TP1
git checkout tp1

# Pour voir le TP2
git checkout tp2

# Pour voir le TP3
git checkout tp3
```

Chaque branche contient un `README.md` spécifique avec les instructions de lancement et le lien vers le **Cours Interactif** (le livrable pédagogique).
